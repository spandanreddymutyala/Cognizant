Bootstrap Directory Structure

1. CSS Folder

The css folder contains Bootstrap's stylesheet files. These files provide the predefined styles for layouts, buttons, forms, navigation bars, tables, cards, and other UI components.

  • bootstrap.css - Readable version for development.
  • bootstrap.min.css - Minified version for production.
  • bootstrap-grid.css - Grid system only.
  • bootstrap-utilities.css - Utility classes only.

2. JS Folder

The js folder contains JavaScript files required for interactive Bootstrap components.

  • bootstrap.js - Full Bootstrap JavaScript.
  • bootstrap.min.js - Minified version.
  • bootstrap.bundle.js - Includes Popper.js for tooltips, dropdowns, and popovers.
  • bootstrap.bundle.min.js - Minified bundle version.

3. Icons Folder

The icons folder contains Bootstrap Icons, a collection of SVG-based icons used in web applications.

  • Provides scalable vector icons.
  • Can be customized using CSS.
  • Useful for buttons, navigation menus, alerts, and dashboards.

Summary Table

Folder Purpose
CSS Contains Bootstrap stylesheets for layout and design.
JS Contains JavaScript files for interactive components.
Icons Contains SVG icons used in Bootstrap projects.
